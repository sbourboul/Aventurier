# Aventurier

### Execution

Launch the Main class with two parameters.  
The first one is the map filename.  
The second is the movements filename.  

**_Example:_**  
`javac Main "/tmp/carte_v2.txt" "/tmp/test1.txt"`

If none of the parameters are entered the test programm will be launched.

**_Example:_**  
`javac Main`
